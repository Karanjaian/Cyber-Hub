# Cyber Hub - Information Security Portal

A 5-page responsive website built for a Web Development assignment, focusing on the core principles of Cybersecurity and Digital Defense.

## 🛡️ Project Overview
Cyber Hub is a technical platform designed to showcase security services, threat analysis, and secure communication. The project demonstrates proficiency in semantic HTML5, CSS3, and responsive web design.

### Key Features
* **5-Page Architecture:** Home, About, Services, Gallery, and Contact.
* **Media Integration:** Embedded YouTube video analysis and high-resolution security-themed imagery.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views using CSS Media Queries.
* **Interactive Elements:** Hover states, CSS transitions, and a functional Contact/Incident Report form.

## 🛠️ Built With
* **HTML5:** Structured content with a focus on accessibility.
* **CSS3:** Custom styles using Flexbox and Grid layouts.
* **Git/GitHub:** Version control and hosting via GitHub Pages.
